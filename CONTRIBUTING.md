# Contributing

Thank you for helping improve the R4FS Forge Calculator. Small, well-supported corrections are especially valuable.

## Reporting data corrections

Open an issue and include:

- The exact equipment or material name.
- The incorrect value or description.
- The proposed correction.
- A reliable source link, screenshot of your own in-game testing, or reproducible test steps.
- Any conditions that affect the result, such as item order, equipment category, inheritance, or skill level.

Keep separate mechanics in separate issues where practical.

## Code contributions

1. Create a focused branch.
2. Keep existing calculator behavior intact unless the change specifically corrects it.
3. Update the appropriate dataset rather than duplicating a material or recipe.
4. Run `npm test` before submitting a pull request.
5. Explain the user-visible effect and testing performed.
6. Update `CHANGELOG.md` for meaningful behavior or data changes.
